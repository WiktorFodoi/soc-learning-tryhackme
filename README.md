# SOC Learning – TryHackMe

## Cel repozytorium
Repozytorium dokumentuje moją naukę roli **SOC Analyst** z wykorzystaniem platformy TryHackMe.
Zawiera notatki, wnioski oraz proste skrypty pomocne w analizie bezpieczeństwa.

## Zakres nauki
- Podstawy SOC (role, procesy, alerty)
- Analiza logów (SSH, HTTP)
- Podstawy sieci
- Typowe ataki (bruteforce, phishing, malware)
- Podstawy Incident Response

## TryHackMe – ukończone / w trakcie
Notatki znajdują się w katalogu `tryhackme/`.

## Narzędzia i technologie
- Linux
- Bash
- grep / awk / sed
- Podstawy sieci (TCP/IP, DNS, HTTP)

## Dlaczego SOC
Interesuje mnie analiza incydentów, praca z logami oraz wykrywanie zagrożeń
w środowiskach produkcyjnych.

## Status
Repozytorium rozwijane na bieżąco w trakcie nauki.
